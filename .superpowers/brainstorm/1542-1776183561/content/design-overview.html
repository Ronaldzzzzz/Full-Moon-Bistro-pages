頁面結構（3頁）

🍽️ 菜單首頁 （公開）
  • 分類頁籤（開胃菜 / 主食 / 甜品 / 飲品 / 套餐）
  • 菜品卡片（圖、名稱、描述、價格、供應狀態）
  • 頁面頂部咖啡廳名稱 Banner
💬 留言板 （公開）
  • 留言列表（ID＋伺服器 或 匿名）
  • 按讚 👍 / 倒讚 👎
  • 回覆（展開顯示，不可互動）
  • 留言表單（下方固定）
⚙️ 後台管理 （密碼保護）
  • 密碼驗證入口（2-3 組密碼）
  • 菜單管理（新增 / 編輯 / 下架）
  • 食材庫存管理
  • 留言管理（刪除）

Firestore 資料模型

menuItems/ { category, name, description, price, imageUrl, available }
inventory/ { name, stock, unit, note }
messages/ { authorId, serverName, isAnonymous, content, timestamp, likes, dislikes }
└─ replies/ { authorId, serverName, isAnonymous, content, timestamp }
adminPasswords/ { hash, role, label }
安全備注：密碼以 SHA-256 雜湊儲存於 Firestore，前端比對雜湊值。Firestore 規則設定菜單與留言公開可讀，寫入需驗證 session token（登入後存於 sessionStorage）。
